Running Instruction:

1. Unzip the file to a location.
2. You will get a folder named "Assignment 1". Change directory to it.
3. Now add the folders "data", "minFunc", "mnist" to it as mentioned on piazza.
4. Now to run each exercise, enter the following commands as following:

Exercise 1:
cd ex1
train

Exercise 2:
cd ex2
train

Exercise 3:
cd ex3
softmaxExercise

Exercise 4:
cd ex4
stlExercise

Note: You need to go back to the main directory "Assignment 1" for each time you want to run a particular exercise.
